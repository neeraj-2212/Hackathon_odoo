How to Set Up and Run This Django Project

clone this project first in any editor and than 

Create a virtual environment:
python -m venv venv

Activate the virtual environment:

On Windows:
venv\Scripts\activate

On macOS/Linux:
source venv/bin/activate

Install required Python package:
pip install Pillow

Run database migrations:
python manage.py migrate

Start the development server:
->Make sure that u are in the folder where environment is being activated and manage.py file is located
python manage.py runserver
