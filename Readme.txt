# ReWear 👕♻️  
A web-based platform to promote **sustainable fashion** by enabling users to **exchange unused clothing** via direct swaps or a point-based redemption system.

## 🌍 Project Overview

**ReWear** encourages users to reuse wearable garments instead of discarding them, helping reduce textile waste and promoting eco-conscious choices.

### 🔁 Key Features
- 🌐 User registration & profile management  
- 📸 Upload clothing items with images, descriptions, and categories  
- 🔄 Direct Swap system: Request item-for-item exchanges  
- 🎯 Point-based Redemption: Earn and spend points to claim clothes  
- 🔍 Browse, filter, and search listings  
- 🛠️ Admin panel for item moderation  
- 💬 Messaging or swap request system (optional add-on)
